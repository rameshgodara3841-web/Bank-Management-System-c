Bank Management System (C)

 Description

This is a simple Bank Management System written in C.
It helps to manage bank accounts using file handling.

 Features

- Create Account
- View Account
- Deposit Money
- Withdraw Money
- Search Account
- Delete Account

⚙️ How to Run

Compile:
gcc bank.c -o bank

Run:
./bank
